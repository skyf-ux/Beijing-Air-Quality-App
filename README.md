# Beijing-Air-Quality-App
A lightweight and interactive Streamlit web application that displays real‑time air quality data for Beijing. Built with Python, the app provides AQI readings, pollutant details, visual charts, and health guidance through a simple and responsive browser‑based interface.

---------------------------------

# 🌍 Air Quality Prediction Dashboard

## 📌 Project Overview
This project predicts PM2.5 levels using machine learning and provides an interactive dashboard for data analysis.

## 🚀 Features
- Data upload & preprocessing
- Exploratory Data Analysis (EDA)
- Data visualisation
- Time series analysis
- Machine learning prediction
- Interactive Streamlit dashboard

## 📊 Technologies Used
- Python
- Streamlit
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📁 Project Structure
- `app/` → Streamlit application
- `models/` → Trained models
- `data/` → Dataset
- `notebooks/` → EDA & model training
- `assets/` → Images for report
